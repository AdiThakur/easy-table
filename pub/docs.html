Welcome to EASYDocs

Here you will find the complete and detailed list of features and functionalities offered by the EASYTable libary. Please visit the getting-started page for setup help, and to get an elemental understanding of the basic usage. For more advanced use-cases, please visit the examples page.

Constructor

Syntax

const myTable = new EasyTable(name, parent_element, options)

Parameters

name

Name of table for easy identification

parent_element

Id of parent element; the newly created table is appended as a child to the element with id parent_element

options

An object-literal that specifies the properties of the table being constructed. The following attributes can be set within the object-literal

columns

Array of strings representing column headers.

defaultStyle [optional]

1, 2, or 3. Sets the corresponding default styles provided by the library

stylesheet [optional]

Path to custom CSS file. Note that each EASYTable DOM element is encapsulated within a shadow-root; Selectors designed for the table will not affect the rest of the HTML.

cssText [optional]

String containing inline-css.

paginate [optional]

An object-literal that specifies the properties of the pagination feature.

perPage

Number of rows per page.

default

Boolean to enable/disable the default buttons for navigating the pages of the table. If set to false, you must explicitly call the .prevPage() and .nextPage() methods to enable navigation between the pages of the table.

defaultSearch [optional]

Boolean to enable/disable the default search bar. If set to false, you must explicitly set the .search() method as a callback to a button-click event (or some alternative) to enable searching.

defaultSort [optional]

Boolean to enable/disable the default sorting buttons. If set to false, you must explicitly set the .sort() method as a callback to a button-click event (or some alternative) to enable sorting.

Return

N/A

Example

const myTable = new EasyTable("myTable", "main", {
    columns: ['Model', 'Brand', 'Year'],
    defaultSearch: true,
    defaultStyle: 2,
    defaultSort: true,
    paginate: { perPage: 4, default: true}
}

Result

Note that some data has been added.

Loading Data-Sets

loadFromCSV()

Syntax

myTable.loadFromCSV(csvString)

Parameters

csvString

String containing data in CSV format. Note that the corresponding columns must be set during instantiation; .loadFromCSV() doesn't set the columns based on data from csvString

Return

true on success, false otherwise.

loadFromJSON()

Syntax

myTable.loadFromJSON(jsonString)

Parameters

jsonString

String containing data in JSON format. Note that the corresponding columns must be set during instantiation; .loadFromJSON() doesn't set the columns based on data from jsonString

Return

true on success, false otherwise.

Row Methods

getRow()

Syntax

myTable.getRow(i)

Parameters

i

Index of the desired row.

Return

The HTMLTableRowElement at index i on success, null otherwise.

setRow()

Syntax

myTable.setRow(i, data)

Parameters

i

Index of the desired row.

data

Array containing the data to be set in the cells of the row at index i. Note that data.length must match the number of columns.

Return

true on success, false otherwise.
