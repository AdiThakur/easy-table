EasyTable Examples Usages

Welcome to the EASYTable demo! In this step-by-step guide, I'll demonstrate how to use the EASYTable library to quickly peform CRUD operations on tables.

In this example, we will be creating a simple three column table that shows us the model, brand, and year of some cars.

Using the EASYTable Libary

We must get the EASYTable library before we can use it. To do so, simply paste the following script tag in your HTML file.

<script defer type="text/javascript" src='EasyTable/EasyTable.js'></script>

If you wish to use one of the three default styles packaged with the libary, you must also incude a tag to get the corresponding stylesheet.

<link rel="stylesheet" href="EasyTable/EasyTable.css">

Creating a New Table

To create a new table, we call the constructor and pass in the following arguments: name of table, id of parent, column headers, enable search, and style.

Using the constructor below, we would create an empty table with the headers Model, Brand, and Year. This table would be appended to the HTML element with id "main", and would use default style 3. Additionally, it would allow users to search its rows (Feel free to try it out in the tables below).

const myTable = new EasyTable("myTable", "main", ['Model', 'Brand', 'Year'], true, 3)

Resulting Table

Adding Some Data

Adding new data is very simple. EASYTable provides two methods for adding a new row: appending and inserting. The first method simply appends the new row to the end of the table, while the second inserts it at a specific spot. Let's add some cars to the table.

Appending Some Rows

myTable.appendRow(["Camry", "Toyota", "2000"])
myTable.appendRow(["Civic", "Honda", "2005"])
myTable.appendRow(["Accord", "Honda", "2001"])
myTable.appendRow(["Mustang", "Ford", "1995"])
myTable.appendRow(["Yaris", "Toyota", "1996"])
myTable.appendRow(["Model-T", "Ford", "1102"])

Inserting a Few More Rows

myTable.insertRow(1, ["Enzo", "Ferrari", "1996"])
myTable.insertRow(1, ["458", "Ferrari", "1996"])

Resulting Table

Updating the Table

Woops! Looks like I misspelled Mustang. Luckily, EASYTable provides methods that allow users to update tables at the granularity of rows OR cells. Realisticaly, these methods would be used to updated specific cells based on certain user interactions, and not to fix spelling mistakes. We can either update the entire row, or just the cell. Both options yeild the same result.

Updating the Entire Row

myTable.setRow(5, ["Mustang", "Ford", "1995"])

Updating Just the Cell

myTable.setCell(5, 0, "Mustang")
