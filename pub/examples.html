EasyTable Demo

Welcome to the EASYTable demo! In this step-by-step guide, I'll demonstrate how to use the EASYTable library to quickly peform CRUD operations on tables (and MORE!)

In this example, we will be creating a simple three column table that shows us the model, brand, and year of some cars.

Using the EASYTable Libary

First, we must get the EASYTable library. To do so, simply paste the following script tag in your HTML file.

<script defer type="text/javascript" src='EasyTable/EasyTable.js'></script>

If you wish to use one of the three default styles packaged with the libary, you must also incude a tag to get the corresponding stylesheet.

<link rel="stylesheet" href="EasyTable/EasyTable.css">

Creating a New Table

To create a new table, we call the constructor and pass in the following arguments: name of table, id of parent, column headers, enable search, and style.

Using the constructor below, we would create an empty table with the headers Model, Brand, and Year. This table would be appended to the HTML element with id "main", and would use default style 3. Additionally, it would allow users to search its rows; the results of the search are the rows that match the combined search query. Feel free to try it out in the tables below.

const myTable = new EasyTable("myTable", "main", ['Model', 'Brand', 'Year'], true, 3)

Adding Some Data

Adding new data is very simple. EASYTable provides two methods for adding a new row: appending and inserting. The first method simply appends the new row to the end of the table, while the second inserts it at a specific spot. Both of the mentioned methods require arrays of data whose length matches the number of columns. Let's add some cars to the table.

Appending Some Rows

myTable.appendRow(["Camry", "Toyota", "2000"])
myTable.appendRow(["Civic", "Honda", "2005"])
myTable.appendRow(["Accord", "Honda", "2001"])
myTable.appendRow(["Gustang", "Ford", "1995"])
myTable.appendRow(["Yaris", "Toyota", "1996"])
myTable.appendRow(["Model-T", "Ford", "1102"])

Inserting a Few More Rows

myTable.insertRow(1, ["Enzo", "Ferrari", "1996"])
myTable.insertRow(1, ["458", "Ferrari", "1996"])

Resulting Table

Updating the Table

Setting Data

Woops! Looks like I misspelled Mustang. Luckily, EASYTable provides methods that allow users to update tables at the granularity of rows OR cells. Realisticaly, these methods would be used to updated specific cells based on certain user interactions, and not to fix spelling mistakes. We can either update the entire row, or just the cell. Both options yeild the same result.

Updating the Entire Row

myTable.setRow(5, ["Mustang", "Ford", "1995"])

Updating Just the Cell

myTable.setCell(5, 0, "Mustang")

Resulting Table

Removing Data

You know what, I no longer want Model-T in our table; its waaaay too old! Once again, as you might have guessed, EASYTable provides multiple easy approaches to the given problem. We can remove the last row by either calling popRow, or deleteRow; popRow always removes from the end of the table, while deleteRow removes a row at a specified index. Calling either of the methods below will remove Model-T from our table.

Popping Row

myTable.popRow()

Deleting Row

myTable.deleteRow(7)

Creating Custom Styles

Users can create numerous styles that are applied to tables upon instantiation. For the Alpha Release, users will be able to use their own custom styles by following the specified naming conventions. To specify a new style for...

  • Rows, create the selector .dataRowX
  • Cells, create the selector .dataCellX
  • Column Headers, create the selector .headerCellX
where X is any integer larger than 3.

For example, if I wanted to create my own style for a table, I could specify the selectors .dataRow4, .dataCell4, and .headerCell4, and pass in the number 4 as the style argument for my table constructor.

Header Selector

.headerCell4 {
 background-color: rgb(0, 0, 0);
 font-size: 1.75rem;
 border: 1px solid #333;
}

Data-Cell Selector

.dataCell4 {
 background-color: rgb(92, 92, 92);
 color: white;
 font-size: 1.25rem;
 padding: 3px;
 border: 1px solid #333;
}

Now, to create a table with this new, custom, style, we pass in 4 as the style argument. Note that some rows have been appended to this table to show off its style, and the search feature has been disabled.

const myTable = new EasyTable("myTable", "main", ['Model', 'Brand', 'Year'], false, 4)
